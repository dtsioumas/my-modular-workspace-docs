## Xiaomi POCO X6 - MIUI Optimization Settings for KDE Connect

### 1. Disable Battery Optimization for KDE Connect:
- Settings → Apps → Manage Apps → KDE Connect
- Battery saver → No restrictions (Important!)
- Background activity → Allow background activity

### 2. Enable Autostart:
- Settings → Apps → Manage Apps → KDE Connect
- Autostart → Enable

### 3. Lock KDE Connect in Recent Apps:
- Open KDE Connect app
- Open Recent Apps view
- Long press or swipe down on KDE Connect card
- Tap the lock icon to prevent MIUI from killing it

### 4. MIUI Optimization (if issues persist):
- Settings → Additional Settings → Developer Options
- Turn OFF "MIUI optimization" (may affect other apps)

### 5. Notification Settings:
- Settings → Apps → Manage Apps → KDE Connect
- Notifications → Enable all notification categories
- Make sure "Persistent notification" is enabled in KDE Connect app

### 6. Permissions:
Grant all permissions KDE Connect requests:
- Files and media access
- Notification access
- Location (for network discovery)
- SMS/Phone (if you want those features)
- Contacts (for contact sharing)

### 7. Data Usage:
- Settings → Apps → Manage Apps → KDE Connect
- Data usage → Allow both WiFi and Mobile data
- Unrestricted data usage → Enable

### 8. Security App Settings:
- Open Security app → Boost speed
- Lock apps → Add KDE Connect
- This prevents the cleaner from killing KDE Connect

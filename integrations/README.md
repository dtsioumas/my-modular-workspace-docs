# Integrations

This directory contains documentation for third-party tools and services and how they are integrated into the modular workspace. The goal is to provide a central place for understanding the architecture and implementation of each integration.

## Documents

- **[KeePassXC Secrets Integration](./keepassxc-secrets.md):** Outlines the architecture for using KeePassXC as the central secret manager for the entire workspace, including shell environments, rclone, chezmoi, and Ansible.
- **[Brave KeepassXC Integration](./brave-keepassxc-integration.md):** Documentation on how to integrate Brave browser with KeepassXC.
- **[Kitty Autocomplete Integration](./kitty-autocomplete-integration.md):** Documentation on kitty terminal autocomplete integration.
- **[Kitty Zellij Integration](./kitty-zellij-integration.md):** Documentation on kitty and zellij integration.
- **[MCP Configuration Architecture](./mcp-configuration-architecture.md):** Document on MCP configuration architecture.
- **[Rclone Bisync Integration](./rclone-bisync-sync-integration.md):** Documentation on rclone bisync integration.
- **[Semantic Search Integration](./semantic-search-integration.md):** Documentation on semantic search integration.
- **[Systemd Keepassxc Secret Management](./systemd-keepassxc-secret-management.md):** Documentation on systemd and keepassxc secret management.
